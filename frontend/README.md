# Frontend Dashboard - Smart People Tracking

React-based web dashboard for monitoring tracked users.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Real-time location tracking on OpenStreetMap (free, no API key required)
- Geofence management
- Location history visualization
- Alert management
- User management (admin)

## Map Technology

This dashboard uses **OpenStreetMap** with **Leaflet** for map visualization. No API keys are required, and the maps are completely free to use. The maps are served by OpenStreetMap tile servers.

### Benefits of OpenStreetMap:
- ✅ Free and open source
- ✅ No API key required
- ✅ No usage limits
- ✅ Community-driven map data
- ✅ Works offline (with proper caching)
